# Domain_Generalization_Analysis

This repository contains a reproduction and analysis of the paper:

Invariant Risk Minimization (Arjovsky et al., 2019) https://arxiv.org/abs/1907.02893

The main objective of this project is to reproduce the core experiments of the paper and study the behavior of Invariant Risk Minimization (IRM) in practice.

This project builds upon the official implementation of IRM:

https://github.com/facebookresearch/InvariantRiskMinimization
